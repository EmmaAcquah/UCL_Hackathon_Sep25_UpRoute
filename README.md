# UCL_Hackathon_UpRoute

## Installation & Usage
### Installation
* Clone repo using `git clone https://github.com/EmmaAcquah/UCL_Hackathon_Sep25_UpRoute.git`
* Run `npm install` for all required dependencies 

### Usage
* Open the terminal and run `npm run dev`
